SubModelStart

The SubModelStart object is used as the entry point for the components of a SubModel.  An entity arriving to a SubModel is directed automatically to its SubModelStart component.

SubModelStart Inputs

Keyword

Description

Key Inputs

 

Description

A free-form string describing the object.

NextComponent

The next component in the sub-model.

Options

 

AttributeDefinitionList, CustomOutputList

See Section 6.2.

Graphics

 

Position, Alignment, Size, Orientation, Region, RelativeEntity, DisplayModel, Show, Movable, VisibleViews, DrawRange

See Section 11.1.

Output Name

Description

Entity and DisplayEntity

 

Name, ObjectType, SimTime, Parent, Attributes, Custom Outputs, Position, Size, Orientation, Alignment, GraphicalLength, ObserverList, NextList, PreviousList

See Section 10.1.
